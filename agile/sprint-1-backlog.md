# Sprint 1 Backlog

Sprint 1 focuses on delivering the core gameplay loop:

```text
scan QR code -> collect item / execute action -> resolve effect -> view inventory
```

Deliver a potentially shippable increment that allows a player to interact with game objects and allows the organizer to resolve action effects.
